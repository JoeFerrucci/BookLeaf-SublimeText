BookLeaf - Scratch Notes Manager
================================

Thanks for installing BookLeaf!

Quick Start:
- Press Cmd+Shift+B then Cmd+Shift+L to open BookLeaf
- Use Command Palette: "BookLeaf: Open"

Commands:
- BookLeaf: Open (Cmd+Shift+B, then Cmd+Shift+L)      - Browse and search your scratch files, create a new file, or save a current unsaved tab to BookLeaf.
- BookLeaf: New File (Cmd+Shift+B, then Cmd+Shift+N)  - Create a new scratch file, or save the current unsaved tab to BookLeaf
- BookLeaf: Search (Cmd+Shift+B, then Cmd+Shift+S)    - Search file contents
- BookLeaf: Delete (Cmd+Shift+B, then Cmd+Shift+D)    - Delete a scratch file
- BookLeaf: Open Folder                               - Open storage folder in Finder

Settings:
- Default extension: .md (configurable)
- Storage: ~/Library/Application Support/Sublime Text/Packages/User/BookLeaf/

To customize settings:
Preferences > Package Settings > BookLeaf > Settings
